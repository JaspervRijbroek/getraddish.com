Features

Just like any other framework Raddish gives compatitors a run for their money, with some stat of the art features for you to use.

Rapid Application Development

The system will do a lot automatically for you,
So you don't have to worry about creating files,
this is only needed for overrides.

Multi database

Raddish supports out-of-the-box Mysql, MongoDB and Sqlite for the database connections.
You don't have to worry about which adapter you use, the system will do that for you and there is no difference in API.

Component Config

We allow you to set basic information into a component config so you don't have to write a complete object to override a simple variable.

Integrated Socket.IO

By having socket.io integrated support for real-time applications are suported.
Also you can use this instance of Socket.IO for your own events and actions.

Threads handler

The system can be set to automatically asses if your system is busy or not and scale appropriately. If it is busy extra threads will be spawned.

Plugins

By making use of plugins you can change the behavior of the system and inject your own functions and methods.

Permissions

Permissions are also handled in the system on the controller level. These will define if you do a certain action.

Using Basics

Raddish relies on the basics of the language. So you can make the methods while using all the basics you know and the smart system of Raddish.

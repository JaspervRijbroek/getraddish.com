Sockets

SocketIO has been integrated into the framework and is completely usable within the system.
In most cases to make the page submit events to SocketIO you only have to add the socketable behavior.

If you want to add your own listeners this is of course possible. This however has to be done in a plugin.
Below an example for this.

{% highlight javascript %} function SocketPlugin() { } SocketPlugin.prototype.onBeforeRegister = function(alias, path) { // The addListener method will add an extra listener function to the sockets. Socket.addListener('demo', function(message) { // We only log the received message but you can do everything here. // Even ObjectManager calls. console.log(message); }); }; module.exports = SocketPlugin; {% endhighlight %}

When sending messages to the clients you can use the Socket.io variable which holds the complete API of SocketIO.
Below an example.

{% highlight javascript %} // Send the "Hello World" message to all connected sockets Socket.io.broadcast('message', 'Hello World'); {% endhighlight %}

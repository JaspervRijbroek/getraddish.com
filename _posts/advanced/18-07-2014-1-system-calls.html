System Calls

Within the system of Raddish you can always call component and core objects.
Using the object ObjectManager you can this object has a method get which accepts an identifier string or an identifier object.

This will return the object related to the identifier.

It is also available to use the ObjectLoader to require files.
The ObjectLoader object is also like the ObjectManager object available everywhere (also outside of objects) and can be used to require an identifier to extend from.

The methods ObjectLoader.require returns an uninitialized object.
Almost identical to the require method, the only difference is that this method will respect the loader sequence.

{% highlight javascript %} var Model = ObjectLoader.require('com://admin/core.model.core'); var util = require('util'); function DemoModel(config) { Model.call(this, config); } util.inherits(DemoModel, Model); module.exports = DemoModel; {% endhighlight %}

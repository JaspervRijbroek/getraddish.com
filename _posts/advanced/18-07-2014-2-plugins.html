Plugins

Plugins give you another level of extensibility.
There are two types of plugins system plugins and application plugins.

System plugins

System plugins work on system events and these are only called in the system.
They extend certain parts of the system.

Usable events:
onBeforeStart, onAfterStart


Application plugins

Application plugins will be called when an application is being registred.
And can change how an application is handled or extend certain functionality.

Usable events:
onBeforeRegister, onAfterRegister

When in the config a different plugins path is defined the system will check this path to require the plugins.
You can place the system plugins in the system folder and the application plugins in the application folder.
The system will loop though all the folders it will find an use the plugins available.

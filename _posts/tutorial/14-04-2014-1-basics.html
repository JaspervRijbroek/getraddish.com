Basics

First of, thanks for taking the time to check out this framework.
Whenever you find a bug, have a request or want to share an idea you are more than welcome to contact me by github.

This Framework

This framework has been created with the intention of a framework which is decoupled and usable for mobile and other devices.
This way we ended up only returning data, which then can be used for different (frontend) frameworks or applications.

In the base this framework is a typical MVC framework with an extension to DCI used in the behaviors.
Together build with s few more behaviors the framework is extraordinary versatile and able to work in most situations.

Internals

The framework is based on identifiers, this are a kind of pointer which will return the object to which the identifier leads to.
for more information about Identifiers you can read here.

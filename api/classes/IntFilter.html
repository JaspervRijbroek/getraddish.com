API - IntFilter

The Integer filter will check if the object is an Integer or not.

Methods

_sanitize (value)

private

The sanitize method will force the value to be an integer.

Parameters
Name Type Description
value * <p>The value to be sanitized.</p>

Returns : <p>The sanitize method will force the value to be an integer.</p>

_validate (value)

private

The validate method will check if the value is a number or not if the value is not a number false will be returned.

Parameters
Name Type Description
value * <p>The value to be validated.</p>

Returns : <p>The validate method will check if the value is a number or not if the value is not a number false will be returned.</p>

getFilter (filter)

This method will examine the cache of the filters and check if the requested filter is present. If not it will try to load this filter, when found it is added to the cache and returned. When there is an error and the filter rejects to load an error is returned.

Parameters
Name Type Description
filter String <p>The name of the filter to get.</p>

Returns : <p>This method will examine the cache of the filters and check if the requested filter is present. If not it will try to load this filter, when found it is added to the cache and returned. When there is an error and the filter rejects to load an error is returned.</p>

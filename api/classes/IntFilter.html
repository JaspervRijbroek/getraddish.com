getFilter (filter)

Defined in lib/filter/filter.js:13

This method will examine the cache of the filters and check if the requested filter is present. If not it will try to load this filter, when found it is added to the cache and returned. When there is an error and the filter rejects to load an error is returned.

Parameters

Name Type Description

The name of the filter to get.

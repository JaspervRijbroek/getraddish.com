API - CoreLocator

The CoreLocator is also part of the ObjectLoader but this objects responsibility is to load core files.

Methods

locate (identifier)

This method will try to locate the core file associated to a "core:..." identifier. If the file is found its containing object is returned, else a false will be returned but when a dependency isn't found the error will be thrown.

Parameters
Name Type Description
identifier ObjectIdentifier The identifier of the file to load.

Returns Boolean/ Object: An object when the file is found else false will be returned.

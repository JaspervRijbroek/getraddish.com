API - ObjectLoader

ObjectLoader object which handles all the loading of the requested objects.

Methods

getSequence ()

This method will check if there is a custom loader sequence, if not return the default.

Returns : <p>This method will check if there is a custom loader sequence, if not return the default.</p>

load (identifier, config)

The load method will check if the object has been cached. If this is not the case it will try to resolve the object.

When an object has been loaded for the first time it is cached for reuse in a later request.

Parameters
Name Type Description
identifier String <p>The identifier of the object to be loaded.</p>
config Object <p>This is the config object to send loaded object.</p>

Returns : <p>The load method will check if the object has been cached. If this is not the case it will try to resolve the object.</p> <p>When an object has been loaded for the first time it is cached for reuse in a later request.</p>

API - Mixin

This is the mixin object, this will help with mixing in the objects.

Methods

has (target)

This method checks if the target has mixins defined or not.

Parameters
Name Type Description
target Object The object to be checked.

Returns Boolean: The value of the hasMixin property.

log (code, message)

This method will log an error message to the defined file.

Parameters
Name Type Description
code Integer The error code to use.
message Message The error message.

mix (target, source)

Mix the mixableMethods of the source into the target. This will also set the hasMixins property on the target.

Parameters
Name Type Description
target Object The object receiving the methods
source Object The object giving the methods

API - RawFilter

The RawFilter accepts any value and always returns true. A RAW value just is the value and whatever the content might be, use with care.

Methods

_sanitize (value)

private

Return the RAW value.

Parameters

value * The value to be sanitzed.

Returns : The RAW value

private

ComponentLocator Class

The ComponentLocator is part of the object loader, this class is responsible for getting the file of a "com://..." identifier.

Defined in: lib/object/locator/component.js:3

Methods

locate

(
  • identifier
)

Defined in lib/object/locator/component.js:14

This method will try to locate the file associated to the identifier, if the file isn't found an error is returned, when a dependency isn't found that error will be thrown.

Parameters

Name Type Description
identifier ObjectIdentifier

The identifier of the file to load.

Boolean/ Object:

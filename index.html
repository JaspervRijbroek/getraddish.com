Raddish NodeJS MVC Framework

Raddish is a new NodeJS Framework incorporating the power of MVC and DCI in a single framework.
Based on output and components this framework is completely decoupled. And perhaps the only framework you will ever need for web applications.

What is Raddish

Raddish is a new type of framework which will allow freedom and the structure of MVC with the power of DCI.

Ridiculous speeds

Powered by promises the framework is Ridiculously fast, Build for geeks

Data centered

The API will only return data, though based for a frontend framework.

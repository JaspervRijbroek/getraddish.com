What is Raddish

Raddis is a backend javascript framework with support for node.js and io.js. Together with the MVC and DCI design patterns it allows for freedom and structure.

Ridiculous speeds

Powered by Promises and Threads the architecture brings amazing speeds and is endlessly scalable. With this it is a powerful base for huge projects.

Data centered

Raddish acts as an API for your rich frontends, so Raddish relies on frontend frameworks like Angular or Ember.

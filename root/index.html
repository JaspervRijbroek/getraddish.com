Amazing speeds

Powered by Promises and Threads the framework is amazingly fast, and endlessly scalable for huge projects which require power above all.

Data driven

The API will only return data therefore greatly based for an API of a frontend framework like Angular or Ember, even a mobile app.

Fully modular

Everything build with Raddish is instantly modular and can be reused whenever and wherever you want it.

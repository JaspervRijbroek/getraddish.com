Features

The most amazing features of Raddish listed together!

Rapid application development

With the atomisation of default tasks your project can be developed faster, and with the usage of these defaults your other developers can adapt to others code faster.

Component config

We allow you to set basic information into a component config so you don't have to write a complete object to override a simple variable.

Integrated Socket.IO

You will be able to use socket.io wherever you want. So you can bind your own events in your own components.

Plugins

This gives you the ability to change behaviors of your system.
